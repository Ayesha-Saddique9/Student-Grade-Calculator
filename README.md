# 🎓 Student Grade Calculator

A simple **JavaScript beginner project** that calculates a student's **total marks, average score, and grade** based on three subject marks.

---

# 🚀 Features

✅ Enter student name

✅ Enter registration number

✅ Enter marks for 3 subjects

✅ Calculate total marks

✅ Calculate average score

✅ Display grade automatically

---

# 🧠 Technologies Used

* 🌐 HTML
* 🎨 CSS
* ⚡ JavaScript (DOM Manipulation)

---

# 📷 Project Preview

The app allows a user to:

1️⃣ Enter student details

2️⃣ Enter subject marks

3️⃣ Click **Calculate**

4️⃣ View results instantly

---

# 📊 Grade Logic

| Average Score | Grade |
| ------------- | ----- |
| 90+           | A+    |
| 80+           | A     |
| 70+           | B+    |
| 60+           | B     |
| Below 60      | Fail  |
